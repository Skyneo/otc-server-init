# otc-server-init

Script to format attached volume, install docker and salt-minion on new Ubuntu or CentOs OTC instances  

Use:

### UBUNTU

./datadisk.sh SIZE  
./ubuntu-python.sh  
./ubuntu-docker.sh  
./ubuntu-saltminion.sh MASTER SALTENV  

### CentOS
./datadisk.sh SIZE  
./centos-saltminion.sh MASTER SALTENV
